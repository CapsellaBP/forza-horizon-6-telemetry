# 配置参数说明

所有参数通过 Web 控制面板调整，保存到 `hud-settings.json`。删除此文件恢复出厂默认。

## 参数索引

### HUD 显示

| 参数 | 默认 | 范围 | 说明 |
|------|------|------|------|
| `hud_opacity` | 0.70 | 0.2-1.0 | HUD 整体透明度 |
| `hud_scale` | 2.0 | 0.3-3.0 | HUD 元素缩放倍数 |
| `rpm_bar_width` | 260 | 100-500 | RPM 条总长 (px) |
| `rpm_bar_brightness` | 1.0 | 0.2-1.0 | RPM 条亮度 |
| `rpm_bar_hue` | 0 | -180~180 | RPM 条色相偏移 (°) |

### 档位/时速字体

| 参数 | 默认 | 范围 | 说明 |
|------|------|------|------|
| `gear_font_size` | 28 | 16-100 | 档位字号 |
| `speed_font_size` | 42 | 16-100 | 时速字号 |
| `label_font_size` | 12 | 8-24 | KM/H SLIP 标签字号 |
| `gear_font_weight` | 500 | 100-900 | 档位字重 |
| `speed_font_weight` | 500 | 100-900 | 时速字重 |
| `gear_italic` | true | - | 档位斜体 |
| `speed_italic` | true | - | 时速斜体 |
| `label_italic` | true | - | 标签斜体 |
| `gear_glow` | true | - | 底部打光效果 |
| `top_row_gap` | 4 | -10~30 | 档位-时速间距 (px) |
| `speed_label_gap` | 3 | 0-20 | 时速-KM/H 间距 (px) |

### 配色

| 参数 | 默认 | 说明 |
|------|------|------|
| `gear_yellow_start` | 0.50 | RPM 条白→黄分界点 (%)。以上黄→橙→红均分 |

### 标记线

| 参数 | 默认 | 范围 | 说明 |
|------|------|------|------|
| `shift_trigger_pct` | 1.0 | 0.8-1.2 | 爆闪触发点 (最优 RPM %) |
| `power_band_pct` | 0.93 | 0.5-1.0 | 功率带阈值 (峰值功率 %) |
| `power_band_opacity` | 0.5 | 0.1-1.0 | 功率带透明度 |
| `marker_ema_alpha` | 0.06 | 0.02-0.50 | 标记线显示层 EMA。越小越稳定 |

### G 力浮动

| 参数 | 默认 | 范围 | 说明 |
|------|------|------|------|
| `g_float_amplitude` | 2 | 0-10 (步进0.5) | 整体浮动幅度 (px) |
| `g_float_smoothing` | 0.08 | 0.02-0.50 | EMA 平滑系数 |
| `g_float_accel_scale` | 0.5 | 0.1-2.0 | 加速方向灵敏度 (加速 G 小可放大) |
| `g_float_brake_scale` | 1.0 | 0.1-2.0 | 刹车方向灵敏度 |
| `g_float_lat_scale` | 1.0 | 0.1-2.0 | 横向 G 力灵敏度 |

### 换挡 (每车)

| 参数 | 默认 | 范围 | 说明 |
|------|------|------|------|
| `shift_aggressiveness` | 0 | -40~40 | 转速偏移 (%)。负=延后，正=提前 |
| `limiter_threshold` | 1.0 | 0.9-1.0 | 断油判定阈值 (最大 RPM %) |

### 滑移

| 参数 | 默认 | 范围 | 说明 |
|------|------|------|------|
| `slip_warn` | 0.10 | 0.01-0.50 | 滑移黄闪阈值 |
| `slip_danger` | 0.50 | 0.05-1.00 | 滑移红色阈值 |

### 曲线采样

| 参数 | 默认 | 范围 | 说明 |
|------|------|------|------|
| `sample_throttle_min` | 0.95 | 0.5-1.0 | 最低油门开度 |
| `sample_skip_frames` | 8 | 0-30 | 换挡后跳过帧数 |
| `curve_alpha` | 0.25 | 1/16~1/2 | EMA 权重。1/16=慢 1/2=快 |
| `power_drop_limit` | 0.0 | 0-1.0 | 同比拒绝阈值。当前功率 < 同转速EMA×此值则拒绝。0=不拒绝 |
| `boost_stable_sample` | false | - | 仅涡轮车。增压稳定后才采样，自吸车无需开启 |
| `boost_stable_frames` | 30 | 5-120 | 增压需连续相同多少帧才确认为稳定基准 |
| `boost_stable_tol` | 0.90 | 0.50-1.00 | 当前增压 >= 基准值 × 此比例即放行采样 |

### 悬挂颜色

| 参数 | 默认 | 范围 | 说明 |
|------|------|------|------|
| `susp_thr1` | 0.30 | 0.10-0.50 | 绿→蓝色边界 |
| `susp_thr2` | 0.55 | 0.30-0.75 | 蓝→黄色边界 |
| `susp_thr3` | 0.80 | 0.55-0.95 | 黄→红色边界 (触底) |

### G 值

| 参数 | 默认 | 范围 | 说明 |
|------|------|------|------|
| `g_total_ema_alpha` | 0.05 | 0.02-0.50 | 总 G 力显示平滑度 |

### EV 检测

| 参数 | 默认 | 范围 | 说明 |
|------|------|------|------|
| `ev_detect_frames` | 300 | 100-2000 | 全油门检测帧数。档位>2 即判为燃油车 |

### 网络

| 参数 | 默认 | 范围 | 说明 |
|------|------|------|------|
| `udp_port` | 5300 | 1024-65535 | UDP 监听端口。修改后需保存并手动重启。游戏内 Data Out 端口需一致 |

## 每车预设

换车时自动保存旧车曲线 + 加载新车曲线。曲线数据、锁定状态随车。存盘文件 `car_curves.json`。

## 存盘文件

| 文件 | 说明 |
|---|---|
| `hud-settings.json` | 所有用户设置 (运行时自动生成) |
| `hud-config.json` | HUD 窗口位置/大小/编辑状态 |
| `car_curves.json` | 每车功率曲线 + 锁定状态 |
